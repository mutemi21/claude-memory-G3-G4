# G3 STM32 Firmware - Complete Codebase Summary

## Product
ECOA G3 PAYG Induction Cooker — STM32G0-based control board firmware with credit/token-based usage, BLE connectivity, OTA updates, and 7-segment display UI.

**Firmware version:** `EKWANG_v6.0.0` (defined in `GlobalDefs/Inc/Definitions.h`)
**Processor:** STM32G071CBUx (Cortex-M0+, 128KB Flash, 36KB RAM, 64MHz PLL clock)
**Build system:** CMake with gcc-arm-none-eabi, C11 standard
**Peripherals used:** I2C1/2, SPI1, USART2/3/4, CRC, RTC, TIM6/7/14/16

---

## Main Loop (Core/Src/main.c)

### Initialization order:
HAL_Init → SystemClock → GPIO/I2C/SPI/UART/CRC/RTC/Timers → W25qxx (SPI flash) → Debug UART → BLE UART → PowerBoard UART → LED → UIView → UIPresenter → UIModel → Keypad → KeypadEvents → DeviceSettings → Flash buffers → SecDataStore → BT power on → IndCooker → CookingSession → Bluetooth → BTStateMachine → Token

### Super loop (runs continuously):
```
UserCommandHandlerExec()    // Debug UART command processing
IndCooker_Process()         // Cooker state machine processing
UIPresenter_Process()       // UI state machine processing
UIView_Process()            // Display rendering
Keypad_Process()            // Keypad scanning
Bluetooth_Churn()           // BLE message processing
Gsm_Churn()                 // GSM/MQTT message processing
```

### Tick (~100ms periodic):
```
Gsm_Tick()
IndCooker_Tick()
Led_Tick()
UIPresenter_Tick()
SessionSummary_Tick()
```

## Key Data Flow

```
Physical buttons → Keypad (I2C scan) → KeypadEvents → UIView (key handler)
                                                          ↓
UIView ←→ UIPresenter (state machine) ←→ UIModel (data)
                    ↓                           ↑
             IndCooker ←→ PowerBoard (UART) ←→ Induction Power Board (hardware)
                    ↓
         CookingSession → SessionSummary → Flash storage
                    ↓
         CreditManagement (deduct credit per cooking time)
                    ↓
Token (BLE/keypad entry) → Nexus keycode validation → Add credit
                    ↓
         Bluetooth/GSM → Cloud (MQTT) → OTA, Shadow, Session uploads
```
